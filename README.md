# Ollama LLM Inference Benchmark

This Python script demonstrates how to interact with a local Ollama server to perform a simple LLM inference. It sends a prompt to a specified model and measures the time taken to receive the response, illustrating a basic benchmarking concept for edge AI devices.

## Language

`python`

## How to Run

1. Ensure Ollama is running locally (e.g., `ollama serve`).
2. Pull a model (e.g., `ollama pull llama2`).
3. Run the script: `python main.py`
